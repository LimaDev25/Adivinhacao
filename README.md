🔮 Projeto 2 – Jogo de Adivinhação (1–100)
Um jogo de adivinhação de números onde o jogador deve descobrir um número secreto entre 1 e 100.

Funcionalidades:

Número aleatório gerado ao carregar a página.

Feedback em tempo real: “muito alto”, “muito baixo” ou “acertou”.

Contador de tentativas.

Campo bloqueado após acerto, com mensagem de conclusão.

🔧 Tecnologias usadas:

HTML5 (formulário e saída de dados)

CSS3 (estilo chamativo com gradientes e animações de entrada)

JavaScript (lógica de validação, comparação e controle do estado do jogo)

🎯 Objetivo:
Exercitar estruturas de controle (if/else, switch), eventos de clique e manipulação dinâmica do DOM em um contexto de jogo simples.
